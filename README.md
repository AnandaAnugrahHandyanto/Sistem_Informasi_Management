# Aplikasi Manajemen Jadwal Mahasiswa (Android Version)

Aplikasi berbasis WebView untuk membantu mahasiswa UNPAM mengelola jadwal kuliah dengan fitur notifikasi otomatis.

## Fitur Utama
- **Login Aman:** Syarat password minimal 8 karakter & huruf kapital.
- **OTP Verification:** Simulasi pengiriman kode ke email.
- **Smart Reminder:** Notifikasi otomatis 10 menit sebelum jadwal kuliah dimulai.
- **Auto-Build CI/CD:** APK di-build otomatis menggunakan GitHub Actions.
